Myplan

Comparador de planes de telefonía móvil para Android

Download this project as a .zip file Download this project as a tar.gz file

Descripción

Control de gasto en llamadas y sms, calculando el importe en función del plan contratado, fecha, hora, duración y el operador del destinatario. Sólo incluye planes de contrato personales, quedando fuera del ámbito del programa los planes para empresas o autónomos y los planes prepago.

Permisos necesarios

  • "LEER SMS O MMS": para contabilizar los mensajes enviados
  • "LEER LOS DATOS DE CONTACTO": para contabilizar las llamadas realizadas.
  • "LEER LA IDENTIDAD Y ESTADO DEL TELÉFONO": para mostrar la ventana de importe al finalizar una llamada.
